# ZabbixApi-Node.Js
Projeto que puxa dados do Zabbix os trata e coloca em uma banco de dados MySql utilizando Node.js e os frameworks Knex, Express e Zabbix-Promise

## Instalar dependencias 

```js
$ npm install
```

## Banco de dados

## Iniando o sistema

```js
$ node index.js
```
